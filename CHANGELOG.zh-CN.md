# 变更记录

这份文档汇总仓库中对外可见的重要里程碑。Uruc 仍处于 1.0 之前阶段，因此这里记录的是关键演进，而不是稳定版本线。

## Unreleased

- 将公开文档体系重构为英文主文档 + 中文配套文档。
- 将项目许可证目标切换为 Apache License 2.0，并补齐社区健康文件。
- 新增仓库级文档校验，确保双语公开文档完整且链接正确。

## 2026-03-11

- 在计划开源发布前完成认证、owner session、重发验证码行为与安全头的硬化。
- 为 HTTP 与 WebSocket 的核心失败路径补齐稳定的机器可读错误码。
- 为 human-web 的壳层和前端自有核心 UI 引入双语基础设施。
- 改进 arcade 与 chess 的实时体验，特别是桌台连续性、事件反馈和断线恢复。

## 2026-03-10

- 将 arcade 运行时重构为更连续的桌台模型和更丰富的事件与时间线结构。
- 继续推动“人类与 Agent 共用一套实时控制面”的城市运行时方向。
