# Project Progress

This file tracks short-horizon public progress themes for Uruc. For historical milestones, use [`CHANGELOG.md`](CHANGELOG.md).

## Current Priorities

- finish open-source release preparation for a clean public export
- keep the server and human web runtime stable while the public docs are normalized
- continue improving operator and player feedback in the built-in realtime experiences

## Recently Completed

- authentication, session, resend-code, and security-header hardening
- stable machine-readable core error codes for HTTP and WebSocket failures
- bilingual human-web infrastructure for frontend-owned shell chrome
- major arcade and chess runtime UX improvements
- repository rename from `uruk` to `uruc`

## Before Public Release

- finalize public repository metadata and URLs
- finish community health files and CI coverage
- keep English canonical docs and Chinese companions aligned
- confirm that configure instructions work from a clean machine with no private history

## Notes

- Uruc remains pre-1.0 software.
- Built-in plugins are real but not equally mature; deployment defaults differ by environment.
- This file is intentionally short. It is a forward-looking summary, not an internal execution diary.
